# Online Quiz Challenge

A responsive, dynamic web-based quiz application built using frontend technologies. The platform allows users to test their knowledge across various topics, track their performance in real-time, and save their high scores under a personalized username.

## 🚀 Features

- **Multi-Page Architecture:** Segregated views for a streamlined user flow (Welcome, Main Menu, Game Arena, Score Summary, and Leaderboard).
- **Dynamic Quiz Engine:** Renders questions dynamically with a progress tracking bar and real-time score updates.
- **Leaderboard System:** Allows users to input their names at the end of a session to save and display scores on a dedicated high-scores dashboard.
- **Responsive Navigation:** Clean top navigation bar for seamless jumping between the platform's core pages (Home, About, Contact).
